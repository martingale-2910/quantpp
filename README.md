# quantpp

A just-for-fun project in C++ on option pricing.

## Features
- **Monte Carlo** option valuation:
    - **European** vanilla option valuation: **Calls, Puts**.

## Requirements 
- C++17
- make

## Building
- **Debug** version (default):
```bash
make
```
or
```bash
make all
```
or
```bash 
make debug
```

- **Release** version (optimized compilation):
```bash
make release
```

## Running:
```bash
make run
```
